# CarRental
Как работник проката, я должен вести реестр авто со следующими данными: марка, госномер, пробег, расход топлива в час, остаток топлива (до 100л), цена минуты, запас хода в часах и общая стоимость аренды. Доступны функции: создание, правка и удаление. Менеджеру важны: общее число машин и список авто с критическим остатком топлива (менее 7 литров).
